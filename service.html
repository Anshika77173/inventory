ADD LOCATION

This HTML document provides a user-friendly interface for adding and displaying location information. It includes a form for inputting location details (name, state, district, pin code, and contact number) which is submitted to `database.php`. An iframe below the form displays existing locations from `display.php`.

The page supports a dark mode, checking localStorage on load to apply the appropriate theme and dynamically updating the theme in response to changes. This ensures a consistent and visually appealing experience for users managing location data.
